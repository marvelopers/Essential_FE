### async vs defer

- async

  - HTML 파싱 시작과 동시에 script를 백그라운드에서 다운로드한 후, 다운로드가 완료되면 HTML 페이지가 파싱 상태와 관계 없이 즉시 실행

- defer
  - HTML 파싱 시작과 동시에 script를 백그라운드에서 다운로드한 후, HTML 페이지 구성이 완료된 다음 script 순차적으로 실행
  - 외부 script에만 유효
